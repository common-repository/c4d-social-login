A simple plugin allows users login by Facebook or Google

== Description ==

This plugin is used in C4D Vision - Responsive  WooCommerce Theme

You can view here http://coffee4dev.com/responsive-woocommerce-theme-c4d-vision/

== Installation ==

1. Upload `c4d-social-login` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

This plugin rely on Facebook App and Google App. 

You need provide Facebook App Id and Google App Id for it can work

https://developers.facebook.com/docs/apps/register

https://developers.google.com/identity/sign-in/web/devconsole-project
